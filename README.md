# wdio-Cermati
## Pre-requisites
- NPM installation https://nodejs.org/en/download
- Node.js >= 14.x
## Prepare Running Project 
- Clone Project
 % git clone https://github.com/ybaha86/wdio-Cermati.git
- Prepare Project
  % npm install
- Run Project
  % npx wdio run ./wdio.conf.js
